你好，欢迎来到这里。

我是一名工作 11 年的程序员，主要在 Linux 环境下从事 C 语言开发工作。  
在长期的项目实践中，也会交替使用 C++、Java、Python、Shell 来解决不同的问题。 

十一年的时间不短，行业和技术都在快速地变，但我对代码的热忱始终没有变。
  
建立这个博客，初衷非常简单，就是想在互联网浮沉之上留下点痕迹。赛博永生？  

这里大抵会用来记录三件事：

 - 对代码的热爱： 记录一些日常的折腾，以及一些个人的编码。

 - 对工作的记录： 记录一些工作中经历的产品与系统，一些踩坑经历和所见所闻。

 - 对生活的纪念： 记录一些自己生活中的思考，以及那些值得留念的瞬间。

如果你恰好路过，无论是技术上的探讨还是生活的共鸣，都欢迎随时交流。
