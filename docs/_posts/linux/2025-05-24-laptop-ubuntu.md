# 笔记本电脑裸机使用ubuntu 22.04

## 背景
1. 朋友把一台淘汰了2018年的`联想拯救者`笔记本电脑送给了我, 因为这台笔记本电脑以前的windows系统是我安装的.  
这台电脑有点邪门, 导致我当时安装系统费了很大劲, 最后曲线救国先安装非官方windows,后使用iso升级到官方windows才成功.  

笔记本信息如下:  
```
CPU: i7 8750H
内存: 16G
硬盘: 256 SSD
显卡: NVIDIA GeForce GTX 1060
```

2. 我驱车来回60公里拿了回来,想着物理机安装fnos, 加上我两个usb硬盘组nas,以便替代迷你主机中的虚拟机fnos(虚拟主系统+直接使用物理磁盘).  

3. 我自己的笔记本电脑是2017年的华硕U4000UQ, 屏幕只有13.6寸. 所以我一直想买一个显示器当副屏. 可是每次打开购物网站商品琳琅满目+攻略七嘴八舌, 就感觉难以下手, 没有看到过无脑入的产品.  
穷, 潜意识里就会总想着能买到性价比高的. 同时,商品的丰富增加了购物难度和购物成本(时间).

### 20260524 系统安装
从早上5:30一直装到8:00, 直到换了三个版本才成功.

| 版本 | 结果 | 原因 |
| --- | --- | --- |
| ubuntu 26.04 | 失败 | 安装中遇到problem, 卡在安装界面到不了最后 |
| ubuntu 24.04.4 | 失败 | 安装中遇到problem, 卡在安装界面到不了最后 |
| ubuntu 22.04.5 | 成功 | 安装成功, 重启后遇到problem, 忽略继续使用 |
| debian 13.6 | 未尝试 | 问ai的话说是系统太新不建议安装 |

结论: 果然电脑有古怪 + 老电脑不宜装新系统.
