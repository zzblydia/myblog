### 2. secureCRT change dir color

1>cd ~  
2>dircolors -p > .dircolors  
3>vi .dircolors  
4>"dir 01;34" ---> "dir 01;35"  
5>source .bashrc

### 3. secureCRT change path color

1>vi .bashrc  
2>env PS1 34 ---> 35  
3>source .bashrc
